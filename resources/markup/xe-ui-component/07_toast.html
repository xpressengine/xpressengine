Toast

toast popup은 사용자 알림의 하나로 일정 시간 이후 자동으로 사라지는 형태이기 때문에 알림의 중요도가 낮거나 사용자 행위를 되돌릴 수 있을 경우에 사용 됩니다.

  • Example
